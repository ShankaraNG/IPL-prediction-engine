About IPL Predictor

This IPL Predictor application is designed to estimate the likely outcome of an Indian Premier League (IPL) match based on historical match data and current match conditions.

Developed by Shankara Narayana N G, this application utilizes a Machine Learning pipeline built using a Random Forest classification model. The model has been trained on IPL match data spanning all seasons from 2008 onwards and considers various match-related parameters such as participating teams, toss outcome, venue, and required run rate to predict the match result.

The primary purpose of this application is to demonstrate the use of Machine Learning techniques in sports analytics and to provide users with an engaging and educational predictive experience.
